PowerShell Saturday

We are considering organizing a PowerShell Saturday (a one-day, small-format technical event from 9am-5pm) in the Chicago area. Before we spend a lot of time and energy organizing the event and making commitments, we want to gauge interest. If you are interested in attending, presenting or assisting with the event (planning/organizing or running it), please join the mailing list below. If we do proceed with organizing the event, we will reach out to those interested via email. Note: it would take about 6 months to organize the event, so the earliest it would happen would be near the end of 2018.
